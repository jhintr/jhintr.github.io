> 案此篇据水野弘元「パーリ語略文法」录入，唯重复处多有省略。

---

#### 1. 以 -an 收尾的阳性词：<span class="box">rājan</span>

|  | 单数 | 复数 |
| :-: | :-: | :-: |
| 第一格 | rāj**ā** | rāj**ā/āno** |
| 第二格 | rāj**aṃ/ānaṃ** | rāj**ā/āno** |
| 第三格 | rañ**ñā**, rāj**inā/ena** | rāj**ehi/ŭhi** |
| 第五格 | rañ**ñā** | rāj**ehi/ŭhi** |
| 第四、六格 | rañ**ño**, rāj**ino/assa** | rañ**ñaṃ**, rāj**ānaṃ/ūnaṃ** |
| 第七格 | rañ**ñe/ñi**, rāj**ini** | rāj**esu/ŭsu** |
| 呼格 | rāj**ă** | -- |

#### 2. 以 -an 收尾的中性词：<span class="box">kamman</span>

|  | 单数 | 复数 |
| :-: | :-: | :-: |
| 第一、二格 | kamm**a/aṃ** | kamm**āni/ā** |
| 第三格 | kamm**anā/unā/ena** | kamm**ehi** |
| 第五格 | kamm**unā** | kamm**ehi** |
| 第四、六格 | kamm**uno/assa** | kamm**ānaṃ** |
| 第七格 | kamm**ani** | kamm**esu** |

- 单数第二格也可以是 kamm**ānaṃ**。

#### 3. 以 -at 收尾的阳性词：<span class="box">bhagavat</span>

|  | 单数 | 复数 |
| :-: | :-: | :-: |
| 第一格 | bhagav**ā/anto** | bhagav**anto/antā** |
| 第二格 | bhagav**aṃ/antaṃ** | bhagav**anto/ante** |
| 第三格 | bhagav**atā/antena** | bhagav**antehi** |
| 第五格 | bhagav**atā** | bhagav**antehi** |
| 第四、六格 | bhagav**ato/antassa** | bhagav**ataṃ/antānaṃ** |
| 第七格 | bhagav**ati/ante** | bhagav**antesu** |
| 呼格 | bhagav**ă/anta/aṃ** | -- |

特例：<span class="box">bhavat</span>

|  | 单数 | 复数 |
| :-: | :-: | :-: |
| 第一格 | bhav**aṃ** | bhav**anto/antā**, bho**nto** |
| 第二格 | bhav**antaṃ**, bho**taṃ** | bhav**ante**, bho**nte** |
| 第三、五格 | bhav**atā**, bho**tā** | bhav**antehi** |
| 第四、六格 | bhav**ato/antassa**, bho**to** | bhav**ataṃ/antānaṃ** |
| 第七格 | bhav**ati/ate** | bhav**antesu** |
| 呼格 | bho, bhan**ta/te**, bhav**ante** | bhan**te**, bhav**anto**, bho**nto** |

#### 4. 以 -at 收尾的现在分词：<span class="box">gacchat</span>

|  | 单数 | 复数 |
| :-: | :-: | :-: |
| 第一格 | gacch**aṃ/anto** | gacch**anto/antā** |
| 第二格 | gacch**antaṃ** | gacch**ante** |
| 呼格 | gacch**aṃ/a/anta** | -- |

- 表中未列部分同以 **-at** 收尾的阳性词。

#### 5. 以 -in 收尾的阳性词：<span class="box">hatthin</span>

|  | 单数 | 复数 |
| :-: | :-: | :-: |
| 第一格 | hatth**ĭ** | hatth**ino/ī** |
| 第二格 | hatth**iṃ/inaṃ** | hatth**ino/ī** |
| 第三、五格 | hatth**inā** | hatth**ĭhi** |
| 第四、六格 | hatth**ino/issa** | hatth**ĭnaṃ** |
| 第七格 | hatth**ini** | hatth**ĭsu** |
| 呼格 | hatth**i** | -- |

#### 6. 以 -in 收尾的中性词：<span class="box">gāmin</span>

|  | 单数 | 复数 |
| :-: | :-: | :-: |
| 第一格 | gām**i** | gām**īni/ī** |
| 第二格 | gām**iṃ/inaṃ** | gām**īni/ī** |

- 表中未列部分同以 **-in** 收尾的阳性词。

#### 7. <span class="box">pitar</span>

|  | 单数 | 复数 |
| :-: | :-: | :-: |
| 第一格 | pit**ā** | pit**aro** |
| 第二格 | pit**araṃ/uṃ** | pit**aro/are** |
| 第三格 | pit**arā/unā** | pit**ūhi/arehi** |
| 第五格 | pit**arā/u** | pit**ĭhi** |
| 第四、六格 | pit**u/uno/ussa** | pit**ūnaṃ/arānaṃ** |
| 第七格 | pit**ari** | pit**ūsu/aresu** |
| 呼格 | pit**ă** | -- |

| 第四、六格 | māt**u/uyā** | māt**ūnaṃ/arānaṃ** |
| 第七格 | māt**ari/uyā/uyaṃ** | māt**ūsu/aresu** |
| 呼格 | māt**ă** | -- |
