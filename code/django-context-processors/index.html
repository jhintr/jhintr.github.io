Django Context Processors

Create a file in your INSTALLED_APPS like profiles/context_processors.py

from django.conf import settings

def site(request):
    return {
    	'SITE_NAME': settings.SITE_NAME,
    }

add this context_processors in your settings.py

TEMPLATES = [
    {
        # ...
        'OPTIONS': {
            'context_processors': [
                # ...
                'profiles.context_processors.site',
            ],
        },
    },
]

then your can use {{ SITE_NAME }} in all templates files.
